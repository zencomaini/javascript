Sejarah CorelDraw

Menurut sumber yaitu dari Wikipedia, CorelDraw menyewa programmer untuk membuat sebuah aplikasi grafik berbasis vekotor yaitu Michel Bouillon dan Pat Beirne. Kemudian aplikasi ini dirilis pada Januari 1989 yang beroperasi pada Sistem Operasi (OS) Window 2.1 versi 1 atau yang dikenal versi pertama 1.0 (satu poin nol) pada saat itu, selanjutnya pada September 1991, CorelDraw merilis versi 1.2 (satu poin dua) pada OS Windows 3.0, semakin dikenalnya CorelDraw oleh para desiner, sehingga CorelDraw terus menambah dan memperbaiki bug pada fitur â€“ fitur nya hingga saat ini, nah saat ini CorelDraw telah memasuki versi CorelDrawX8 atau disebut juga versi ke 18.
Ada yang unik dari CorelDraw, lebih banyak mengeluarkan versi terbaru berbasis OS Windows ketimbang OS lainnya seperti Linux, namun pada Mac OS sendiri CorelDraw tidak lagi bisa digunakan karena alasan kurangnya peminat ketimbang OS Windows, pada Linux sendiri CorelDraw telah merilis versi 11 tahun 2001.
Nah itu adalah sejarah singkat tentang CorelDraw, jika kalian berminat untuk belajar CorelDraw, maka gunakanlah CorelDraw yang secara resmi ya, jangan mendownload sembarangan CorelDraw karena kemungkinan ada virus yang di masukkan kedalam package, sehingga akan merusak Sistem Operasi pada computer atau laptop Kalian.

Tools CorelDraw

  1. Pick Tool

    2. Pick Tool berfungsi untuk menyeleksi objek yang sedang di desain. Dan juga dapat membuat atau merubah bentuk objek, sesuai dengan keinginan

  2. Shape Tool

    3. Shape tool mempunyai fingsi merubah objek yang di inginkan, dan bertumpu pada titik sudut pada objek tersebut. Sudut dapat di ubah menjadi bulat atau runcing, sesuai kebutuhan.

  3. Smooth Tool

    4. Tool ini berfungsi untuk merapikan atu menghaluskan, bahkan meluruskan objek yang kalian pilih

  4. Smear Tool

    5. Smear Tool biasanya digunakan untuk merubah objek menjadi melengkung, dimana cara kerjanya kalian harus menarik baik kearah dalam ataupun arah luar objek.

  5. Twirl Tool

    6. Tool ini berfungsi untuk merubah objek yang bisa berputar, dengan cara klik bagian kotak keci pada bagian objek, dan tahan hingga membentuk objek lingkaran sesuai keinginan.

  6. Attract and Repel Tool

    7. Tool ini adalah tool yang bebas berekpresi, yang dimaksud, kalian bisa mengubah objek sesuai keinginan, nah tool ini paling sering digunakan pada pembuatan ilustrasi, sketsa dan gambar kartun.

  7. Smudge Tool

    8. Tool yang berfungsi mengubah objek dengan bentuk lengkungan, dimana tool ini berguna ketika membuat ilustari Puzzle.

  8. Roughen Tool

    9. Tool yang unik satu ini, bisa kalian gunakan dalam membuat ilustrasi yang rumit, pada tool ini m=berfungsi mengubah bagian objek bergerigi.

  9. Crop Tool

    10. Tool ini namanya hampir sama disemua aplikasi desain, bahkan di Smartphone pun ada fitu ini. Fungsinya adalah untuk memotong objek dengan detail.

  10. Zoom Tool

    11. Zoon Tool ini biasanya digunakan oleh desainer, untuk memperbesar objek, dimana pada objek tersebut biasanya mereka mengedit bagian terkecil. Bahkan pada desainer, untuk membuat objek terkecil dari objek awalnya, menggunakan tool ini.
